# DeFi Sentinel Hackathon Project

This repository contains the source code for the DeFi Sentinel project, prepared for the hackathon submission.

The project is divided into three main components:

1.  [**Backend**](./defi-sentinel-backend) - Node.js/TypeScript backend services.
2.  [**Frontend**](./defi-sentinel-frontend) - Next.js web application.
3.  [**Smart Contracts**](./defi-sentinel-smart-contracts) - Hardhat project for smart contracts.

## Getting Started

Please see the README files in each subdirectory for specific instructions on how to set up and run each component.

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm or yarn
